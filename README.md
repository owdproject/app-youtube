## Overview

This app for Open Web Desktop provides a YouTube and YouTube Music player.

## Installation

```bash
nx run desktop:install-app --name=@owdproject/app-youtube
```

## Usage

### Available commands

```
youtube <youtube-url>
youtube --new
youtube --music <youtube-url>
```
